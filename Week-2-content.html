Cognitive Processes: Attention & Perception

Attention

Attention is the mental process of selecting a limited amount of information from the many stimuli present in the environment. Because human cognitive capacity is limited, users cannot attend to everything at once.

Perception

Perception is the process of interpreting and giving meaning to sensory information based on previous knowledge and experience.

Short-Term Memory (STM) vs Long-Term Memory (LTM)

Memory affects how users interact with systems. Interfaces should reduce reliance on short-term memory and support recognition over recall.

Comparison of STM and LTM
Aspect Short-Term Memory Long-Term Memory
Capacity Limited (7 ± 2 items) Very large
Duration Short Long-lasting
Design Implication Avoid overload Use familiar concepts

Forgetting Curve

The forgetting curve shows how information is lost over time when there is no attempt to retain it.

  • Forgetting is rapid initially
  • Memory loss slows over time
  • Revision improves retention

Norman’s Seven Stages of Action

Donald Norman’s model explains how users form goals, execute actions, and evaluate outcomes during interaction.

  1. Form the Goal
  2. Form the Intention
  3. Specify the Action
  4. Execute the Action
  5. Perceive the System State
  6. Interpret the System State
  7. Evaluate the Outcome

Gulf of Execution & Gulf of Evaluation

Gulf of Execution

The gap between what a user wants to do and the actions the system allows.

Gulf of Evaluation

The gap between the system’s output and the user’s understanding of it.

Comparison of the Two Gulfs
Gulf Problem Design Goal
Execution Hard to perform actions Make actions clear
Evaluation Hard to interpret feedback Provide meaningful feedback

Case Study: Automatic Syringe Interface

A numeric keypad syringe interface allowed dangerous dosing errors due to slips in number entry.
