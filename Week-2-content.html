Week 2 HCI Notes

1. Cognitive Processes: Attention & Perception

Attention

Attention is the mental process of selecting a limited amount of information from the many stimuli present in the environment. Because human cognitive capacity is limited, users cannot attend to everything at once.

In interface design, attention determines what users notice first and what they may completely miss.

Perception

Perception is the process of interpreting and giving meaning to sensory information such as visuals or sounds, based on previous knowledge and experience.

Good interfaces support correct perception by using clear layouts, familiar icons, and consistent visual cues.

2. Short-Term Memory (STM) vs Long-Term Memory (LTM)

Memory strongly affects how users interact with systems. Interfaces should reduce the load on short-term memory and rely on recognition instead of recall.

Comparison of STM and LTM
Aspect Short-Term Memory (STM) Long-Term Memory (LTM)
Capacity Limited (around 7 ± 2 items) Very large / unlimited
Duration Few seconds Long-lasting
Design Implication Avoid overloading users Use familiar concepts

3. Forgetting Curve

The forgetting curve explains how information is lost over time when there is no attempt to retain it. Forgetting is rapid at first and then slows down.

  • Fast memory loss occurs in the beginning
  • Forgetting slows over time
  • Revision helps retain information

4. Norman’s Seven Stages of Action

Donald Norman proposed the Seven Stages of Action to describe how users interact with systems and where usability problems occur.

  1. Form the Goal
  2. Form the Intention
  3. Specify the Action
  4. Execute the Action
  5. Perceive the System State
  6. Interpret the System State
  7. Evaluate the Outcome

5. Gulf of Execution & Gulf of Evaluation

Gulf of Execution

The Gulf of Execution is the gap between what the user wants to do and the actions provided by the system.

Gulf of Evaluation

The Gulf of Evaluation is the gap between the system’s output and the user’s understanding of whether the goal was achieved.

Gulf Comparison
Concept Description Design Goal
Execution Difficulty performing actions Make actions clear
Evaluation Difficulty understanding feedback Provide meaningful feedback

6. Activity: Norman’s Action Cycle – Automatic Syringe Interface

The automatic syringe used a numeric keypad to enter doses. A single key slip could result in a ten-times overdose, making it a dangerous interface.

Applying Norman’s Action Cycle
Stage Bad Design (Numeric Keypad) Good Design (Digit Buttons) HCI Principle
Goal Deliver correct dose Deliver correct dose Safety-critical task
Plan Remember exact number Adjust digits gradually Reduced cognitive load
Specify Type numeric sequence Increase/decrease digits Recognition over recall
Execute Single slip causes overdose No large jumps possible Error prevention
Perceive Raw numbers only Clear digit control Visibility
Interpret Confusing feedback Easy to understand Clear feedback
Compare Difficult to verify Easy to confirm Supports evaluation
