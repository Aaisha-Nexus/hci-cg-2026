Cognitive Processes: Attention & Perception

Attention

Attention is the mental process of selecting a limited amount of information from the many stimuli present in the environment. Users cannot attend to everything at once.

Perception

Perception is the process of interpreting and giving meaning to sensory information based on previous knowledge and experience.

Attention vs Perception
Process Definition Example in Interfaces
Attention Selecting limited info from environment Side panel menus show only relevant options
Perception Interpreting sensory info using prior knowledge Icons and labels interpreted correctly by users

Short-Term Memory (STM) & Long-Term Memory (LTM)

Memory affects how users interact with systems. Interfaces should reduce reliance on STM and support recognition over recall.

STM vs LTM Comparison
Aspect Short-Term Memory Long-Term Memory
Capacity Limited (7 ± 2 items) Very large
Duration Short Long-lasting
Design Implication Avoid overload, chunk info Use familiar concepts, categories

Forgetting Curve

The forgetting curve shows how information is lost over time without review.

Key Points of Forgetting Curve
Observation Implication for Interface Design
Rapid initial forgetting Provide immediate reminders or tooltips
Memory loss slows over time Schedule spaced repetition or tutorials
Revision improves retention Allow repeated access to key info

Norman’s Seven Stages of Action

Donald Norman explains how users form goals, execute actions, and evaluate outcomes.

Stages of Action
Stage Description
Form GoalUser decides what to achieve
Form IntentionUser plans how to reach the goal
Specify ActionUser determines exact actions
Execute ActionUser carries out actions
Perceive System StateUser observes system response
Interpret System StateUser understands system feedback
Evaluate OutcomeUser compares result to goal

Gulf of Execution & Evaluation

Comparison of the Two Gulfs
Gulf Problem Design Goal
Execution Hard to perform actions Make actions clear
Evaluation Hard to interpret feedback Provide meaningful feedback
